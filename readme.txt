== Description ==

TrackTech.io is a tracking system for ecommerce shops. You need a account at tracktech.io. There is a 30 days trail version of every plan and a free version as well. Check out tracktech.io for more info.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tracktech` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings to configure the plugin
